# Design Resources

## On-demand reading

This skill is loaded on demand — not globally auto-imported. Load it when:

- The designer asks "where do I learn X" or "show me how good products handle Y".
- Inspiration is needed for a UI pattern.
- The designer asks for platform-specific guidelines.

---

## Live browse → capture → translate workflow

When the designer asks "show me how good products do X" or "find inspiration for Y":

1. **Open Mobbin** (or another gallery) using Claude in Chrome.
2. **Capture 3–5 real screens** that demonstrate the pattern.
3. **Summarize the pattern** — what makes these examples effective?
4. **Translate to the project's UI library** — how would this look using shadcn/ui / MUI / Ant Design / Chakra / local components?

Inspiration is an **idea, not a spec** — translate patterns, not pixels. Never copy a competitor's exact visual design.
